# MyLocationsAppSwiftUI

My Locations app in SwiftUI with MVVM and MapKit

 
 ### Features
 
 	 • SwiftUI
  
 	• Project Organization & Architecture	
	
	• MVVM 
	
 	• Support iPhone & iPad
	
 	• Portarit & Landscape

	• Custom Animations effect & transition
